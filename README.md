# devOps_lab1_weather
Требования:
1. Написать bash-скрипт, который принимает в качестве входящего параметра город.
2. Выводит температуру и влажность в текущий день в этом городе.
3. Установить nginx.
4. Скрипт запускать по крону раз в минуту, вывод сохранять в index.html дефолтного сайта.

**Содержимое скрипта:** update-weather.sh

**Расписание выполнения:** каждую 1 минуту
